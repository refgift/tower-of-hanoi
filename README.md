# Tower of Hanoi, by grok-code-fast 

It uses the curses library, you might need to install ncurses-devel or equivalent to get it to make.
This is by far the best tower of hanoi test I have made with a LLM coder. 
I use it as a test because it was a benchmark
for the FORTH language many many years ago.

It runs in the terminal.

This is the program that decided me for staying with grok and opencode for my software development. It showed me
the potential of grok-code-fast-1. I used the opencode cli to get it to work. Grok told he programs
in his mind and has no sandbox for C programs, only python.
